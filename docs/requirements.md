# Systemanforderungen - Übersicht

Diese Seite bietet eine Übersicht über die Systemanforderungen.

## Anforderungstypen

### [Customer Requirements (CR)](./customer-requirements.md)
Funktionale Anforderungen aus Anwendersicht:
- Überwachung von pH, EC, Temperatur
- Kalibrierung und Messgenauigkeit
- Multi-Node-Unterstützung
- Datenexport und Visualisierung

**16 Anforderungen** (CR-01 bis CR-16) mit Prioritäten: MUSS / SOLL / KANN

### [Technical Requirements (TR)](./technical-requirements.md)
Technische Umsetzungsanforderungen:
- Systemarchitektur (Node-Hub-Modell)
- Firmware (ESP32, Sensoren, Kalibrierung)
- Backend (API, Datenbank, WebSocket)
- Frontend (UI, Visualisierung)
- Datenmodell und Protokolle

**28 Anforderungen** (TR-01 bis TR-41) kategorisiert nach Komponenten
